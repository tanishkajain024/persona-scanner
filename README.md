# Persona Scanner 🧠

## 📌 Description
Persona Scanner is a Python-based project that analyzes a user's personality using:
- Numerology (Life Path Number)
- Simple personality questionnaire

## 🚀 Features
- Calculates Life Path Number from birthdate
- Conducts personality quiz
- Generates a personality report
- User-friendly menu system

## 🛠️ Tech Stack
- Python

## ▶️ How to Run
1. Clone the repository
2. Run the file:

   python app.py

## 📊 Output
- Personality insights based on numerology
- Behavioral traits from quiz
- Combined personality report

## 🎯 Purpose
This project is built for educational purposes to demonstrate basic personality analysis logic.
